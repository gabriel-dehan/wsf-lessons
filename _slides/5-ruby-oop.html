méthode initialize de la classe `User` @admin = true end end admin = Admin.new({ first_name: "Gabriel", last_name: "Dehan", age: 24 }) #=> # ``` --- ## Modules Un module ou *namespace* permet de mieux organiser son code et d'éviter les conflits. -- ### Définition * \- On utilise le mot clef `module` pour définir un module * \- On accède à une classe du module via `NomDuModule::NomDeLaClasse` ```ruby module Model class User # ... end class Post # ... end end module Controllers class User end class Post end end ``` ```ruby # Model::User, Model::Post, Controllers::User, Controllers::Post user = Model::User.new({ first_name: "Gabriel", ...}) user_controller = Controllers::User.new({ model: user, view: ... }) ``` -- ### Exemples du monde réel ```ruby #
